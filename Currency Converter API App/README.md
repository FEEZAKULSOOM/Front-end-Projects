# Currency Converter API App

A real-time Currency Converter web application built with Vanilla JavaScript and powered by a live Exchange Rate API.  
The app converts currencies instantly with a clean, professional, and responsive user interface.

## Features
- Real-time currency conversion using an external API
- Supports multiple international currencies
- Dynamic country flag updates based on selected currency
- Automatic dark mode (system-based)
- Clean, modern, and responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- ExchangeRate API

## API
ExchangeRate API  
https://www.exchangerate-api.com/

Example endpoint:
https://v6.exchangerate-api.com/v6/YOUR_API_KEY/latest/USD


## How It Works
1. Enter an amount
2. Select source and target currencies
3. The app fetches live exchange rates from the API
4. The converted amount is calculated and displayed instantly
